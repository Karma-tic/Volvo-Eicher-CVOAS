# Volvo–Eicher Commercial Vehicle Operations & Analytics System

## Backend (ERP Core)

This backend simulates an ERP-style system inspired by commercial vehicle manufacturers like Volvo–Eicher.

### Modules Implemented
- Plant Master
- Part Master
- Inventory Stock
- Stock Movement (IN / OUT with audit trail)

### Technology Stack
- Node.js
- Express.js
- MongoDB (Atlas)
- Mongoose ODM
- REST APIs

### Key ERP Concepts Covered
- Master data vs transactional data
- Inventory control
- Stock audit trail
- Data integrity and validation
